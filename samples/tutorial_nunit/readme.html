Installation

  1. Build the NUnit addin using Visual Studio 2010.
  2. Copy NUnitSquishCoco.dll to the addin folder of NUnit.

Setup

Edit the build.bat script and set the NUNIT to the path of the NUnit main installation folder.

Execution

The sample is performing following steps:

  1. Compilation of tutorial_nunit.dll with Squish Coco's compiler wrapper:
    after the build, the instrumentation database tutorial_nunit.dll.csmes is generated.
  2. Execution of the unit tests using nunit-console.exe:
    during the execution of the tests, the execution report tutorial_nunit.dll.csexe is generated for each single unit test.
  3. Import of the execution report into the instrumentation database:
    the instrumentation database is now containing the code coverage information for all unit tests.
  4. Open the instrumentation database with CoverageBrowser.
