Instrumenting The C# Library

Instrument the C# Tutorial.DLL library by adding COVERAGESCANNER_COVERAGE_ON to the conditional compilation symbols. The test DLL may not be instrumented.

Recording The Code Coverage for Each Unit Test

To record the code coverage for each single unit test it is necessary to modify the tests as followings:

  • The unit test should be inherit the class Coverage
  • Add to the member TestContext the attribute override

The Coverage class will record the code coverage of each C# unit test for the instrumented code which is executed. Its code is provided as sample in the file coverge.cs and should be adapted to the projects needs.

Note:  If an unit test does not inherit Coverage, then all coverage informations are stored in one execution report called "unknown".
